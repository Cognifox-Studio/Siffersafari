# Security Policy

## Supported Versions

Aktiv utvecklingsgren:

- `main`

## Reporting a Vulnerability

Rapportera säkerhetsproblem privat via en GitHub Issue med etiketten `security` och skriv endast en kort sammanfattning offentligt.

För känsliga detaljer, kontakta maintainer direkt och inkludera:

- Hur sårbarheten kan reproduceras
- Påverkan/risk
- Förslag på åtgärd (om du har)

Vi bekräftar mottagning så snabbt som möjligt och prioriterar verifierade säkerhetsproblem före vanliga buggar.

Tack för att du hjälper till att hålla Siffersafari säkert.
