# Code of Conduct

## Our Pledge

Vi vill ha en respektfull, inkluderande och trygg samarbetsmiljö för alla som deltar i projektet.

## Expected Behavior

- Visa respekt i diskussioner och feedback
- Håll fokus på sakfrågor, kod och förbättringar
- Var tydlig, hjälpsam och konstruktiv

## Unacceptable Behavior

- Personangrepp, trakasserier eller hot
- Hatiskt, diskriminerande eller kränkande innehåll
- Medvetet störande beteende i issues, PR:er eller diskussioner

## Enforcement

Maintainers kan moderera innehåll, låsa trådar eller begränsa deltagande vid överträdelser.

## Scope

Denna uppförandekod gäller i hela repot: issues, pull requests, code reviews och diskussioner.
