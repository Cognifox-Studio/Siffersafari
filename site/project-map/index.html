Live Project Map

Hela appens struktur, i en uppdaterad karta.

Sidan läser den genererade grafen från graph-data.js och ritar en modulöversikt för appstart, skärmar, tillstånd och tjänstkoppling, tjänster och resten av fundamentet. Kör generatorn i watch-läge så uppdateras kartan löpande när du ändrar Dart-kod.

Välj en modul

Klicka på en nod i kartan för att se dess filer, roller, beroenden och vilka andra delar av appen som använder den.

Hur du använder sidan
1. Generera kartan
`dart run scripts/generate_project_map.dart --watch`
2. Öppna sidan
Öppna filen `site/project-map/index.html` i webbläsaren.
3. Följ förändringar live
Sidan hämtar om datafilen automatiskt och ritar om grafen vid ändringar.
