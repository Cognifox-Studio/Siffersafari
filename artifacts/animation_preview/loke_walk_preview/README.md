# Loke Walk Preview

Denna preview visar en lokal browser-animation för Loke utan emulator eller Flutter-körning.

## Lokal körning

1. Starta en enkel server i repo-roten, till exempel `py -m http.server 8765`.
2. Öppna sedan `http://127.0.0.1:8765/artifacts/animation_preview/loke_walk_preview/` i webbläsaren.

Previewn är en snabb visuell validering av walk-rörelsen, inte en färdig Rive-export eller runtime-asset.
