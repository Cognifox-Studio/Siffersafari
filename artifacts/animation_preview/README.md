# Animation Preview Lab

Denna katalog (rtifacts/animation_preview/) används för att visualisera, testa och isolera karaktärsanimationer utanför appens huvudkod.

## Archived / Research

- loke_walk_preview/ — Research och proof of concept, utgjorde bas för karaktärssystemet.
- skogshjalte_*_preview/ — Tidiga development artifacts, existerar inte i produkten.
- skog_loke_forest_log_celebrate_preview/ — Experimentell kombinerad scen.

## Active (Production-relevant)

- mascot_walk_preview/ — Huvudsaklig referens och QA-miljö för Mascot runtime.
- pivot_reference_preview/ — Generisk animationsreferens för svängar och byte av riktning.

## Uppdateringar & Framtid

När nya animationer eller riggar byggs, ska de först verifieras i motion-labs här.
Se .github/skills/animation-preview-lab/SKILL.md för detaljerat workflow och riktlinjer kring hur previews skapas och godkänns.
