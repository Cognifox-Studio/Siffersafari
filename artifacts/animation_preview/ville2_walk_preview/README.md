# Ville 2 Walk Preview

Denna preview laddar samma Lottie-fil som ligger i `assets/animations/ville2_walk_cycle.json`.

## Lokal körning

1. Starta en enkel server i repo-roten, till exempel `py -m http.server 8765`.
2. Öppna sedan `http://127.0.0.1:8765/artifacts/animation_preview/ville2_walk_preview/` i webbläsaren.

Previewn är bara till för visuell validering utanför emulatorn. Själva källan för animationen är JSON-filen i `assets/animations/`.
