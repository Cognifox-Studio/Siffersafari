# Siffersafari 1.3.2

## Features
- **Nivåuppgångs-celebration**: En festlig banner visas på resultatskärmen när barnet passerar en ny nivå. 🎉
- **SRS-status i resultatskärmen**: Om spaced repetition är aktiverat visas hur många frågor som sparats för framtida repetition.
- **Streak-reset-feedback**: Om barnets daily challenge-streak bryts visas ett uppmuntrande meddelande om att starta en ny.

## Improvements
- Onboarding: Om ett barn startar utan att ha valt årskurs sätts åk 1 automatiskt som standard.
- Analytics: `level_up`-event loggas med `old_level`, `new_level` och `title` vid varje nivåuppgång.
