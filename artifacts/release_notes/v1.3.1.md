# Siffersafari 1.3.1

## Features
- **Daily Challenge streak**: Appen spårar nu hur många dagar i rad barnet slutfört daglig utmaning. 🔥-badge visas på hemskärmen när streak > 1 dag.
- **Combo-multiplikator**: 1.5× poängbonus vid 3+ rätt i rad, 2.0× vid 5+ rätt. Visas som orange badge i feedback-dialogen.
- **Personaliserad daglig utmaning**: Dagens utmaning väljs nu baserat på barnets mastery och svårighetssteg per räknesätt, inte bara kalender.
- **Story landmark-hints**: Storykartans `NowAndNextPanel` visar platsbeskrivning för aktuell checkpoint.

## Improvements
- Stavningsfix i hemskärmen (`Fortsätt`, `är redo`).
- Förbättrad providerstruktur för dagliga utmaningar (family per userId).
