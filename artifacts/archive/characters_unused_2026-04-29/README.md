# Arkiverade Legacy-karaktärer (2026-04-29)

Karaktärerna Loke, Skogshjalte och Ville var experiment och tester under en tidigare iterativ fas (våren 2026) för att hitta rätt asset- och animationsworkflow. 

Efter beslutet att satsa 100% på procedurgenererade composit-SVGs för Mascot-runtimen (SVG-first) och att Rive är ett isolerat research-spår, togs dessa bort från build-kontexten för att minska repots vikt och onödig förvirring.
